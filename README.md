### 关键参数

* 200 MHz Cortex-M4F
* 192KB SRAM + 512KB FLASH
* 4xI2S，内置音频 PLL支持音频级采样精度
* 12bit 2.5MSPS ADC
* USB 2.0 FS
* 192KB SRAM, 512KB Flash
* 200Mbps XIP QSPI
* 可编程增益放大器 PGA
* QFN32,25个GPIO
