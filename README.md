#### 关键特性

* 4xI2S，内置音频 PLL支持音频级采样精度
* 12bit 2.5MSPS ADC
* USB 2.0 FS
* 192KB SRAM, 512KB Flash
* QSPI: 200Mbps XIP
* 3xUSART
* 可编程增益放大器（PGA）
